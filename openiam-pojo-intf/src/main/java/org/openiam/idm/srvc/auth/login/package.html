The diamelle.security.auth package consists of entity and session beans that are related to the authentication of the user and the enablement of single sign on at the API level..

 Authenticator

This component authenticates the user taking into account the appropriate form of authentication and supporting policies and authentication workflow that are in effect.
SecurityManager

This component provide authorization capability which includes:

  • Determining authorization level based on rules and policies
  • Managing access to groups and roles
  • Managing access to applicaitons menu options

 
Group Every user is associated to one or more groups. Each group has its accesses, entitlements, policies and permissions. The Group component provides functionalities to add a new group, update the existing ones and remove the group.
UserManager This component can be used for managing user profiles. It has the functionality to create new user record, update the record of existing user, search user using search property, login for an existing user. Every user belongs to one or more groups. This component allows the additional features to assign groups to user, remove user from groups, get all groups to which the user belongs, lists all users for a particular group. 
AuthState This component that maintains the authenticated state of the user across all the systems they are accessing in the enterprise.
